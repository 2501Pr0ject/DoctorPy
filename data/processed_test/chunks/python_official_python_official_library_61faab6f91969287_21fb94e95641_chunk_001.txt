None False True __debug__ SyntaxError ## Constants added by the site module¶ site The site module (which is imported automatically during startup, except if the -S command-line option is given) adds several constants to the built-in namespace. They are useful for the interactive interpreter shell and should not be used in programs. site -S Objects that when printed, print a message like “Use quit() or Ctrl-D (i. e. EOF) to exit”, and when called, raise SystemExit with the specified exit code. SystemExit Object that when printed, prints the message “Type help() for interactive help, or help(object) for help about object. ”, and when called, acts as described elsewhere. elsewhere Objects that when printed or called, print the text of copyright or credits, respectively.
