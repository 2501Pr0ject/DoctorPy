random() ## Command-line usage¶ Added in version 3. 13. The random module can be executed from the command line. random python -m random [-h] [-c CHOICE [CHOICE. ] | -i N | -f N] [input. ] The following options are accepted: Show the help message and exit. Print a random choice, using choice(). choice() Print a random integer between 1 and N inclusive, using randint(). randint() Print a random floating-point number between 0 and N inclusive, using uniform(). uniform() If no options are given, the output depends on the input: String or multiple: same as --choice. String or multiple: same as --choice. --choice Integer: same as --integer. Integer: same as --integer. --integer Float: same as --float. Float: same as --float. --float ## Command-line example¶ Here are some examples of the random command-line interface: random $ # Choose one at random $ python -m random egg bacon sausage spam "Lobster Thermidor aux crevettes with a Mornay sauce" Lobster Thermidor aux crevettes with a Mornay sauce $ # Random integer $ python -m random 6 6 $ # Random floating-point number $ python -m random 1. 8 1. 7080016272295635 $ # With explicit arguments $ python -m random --choice egg bacon sausage spam "Lobster Thermidor aux crevettes with a Mornay sauce" egg $ python -m random --integer 6 3 $ python -m random --float 1. 8 1. 5666339105010318 $ python -m random --integer 6 5 $ python -m random --float 6 3. 1942323316565915 ### Table of Contents random — Generate pseudo-random numbers Bookkeeping functions Functions for bytes Functions for integers Functions for sequences Discrete distributions Real-valued distributions Alternative Generator Notes on Reproducibility Examples Recipes Command-line usage Command-line example random Bookkeeping functions Functions for bytes Functions for integers Functions for sequences Discrete distributions Real-valued distributions Alternative Generator Notes on Reproducibility Examples Recipes Command-line usage Command-line example #### Previous topic fractions — Rational numbers fractions #### Next topic statistics — Mathematical statistics functions statistics ### This page Report a bug Show source random random() 0. 0 <= X < 1.
